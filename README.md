LinguaSign AI: Real-Time Sign Language Translator 🤟

❗ REQUIRED STEP BEFORE RUNNING

You MUST unzip the Hackathon\_Demo\_Data.zip file into the main folder. The application expects a folder named Hackathon\_Demo\_Data to be present to load the vocabulary.



🚀 The Mission

LinguaSign AI is a Deep Learning prototype built to break communication barriers. Using computer vision and high-performance Neural Networks, it translates Malaysian Sign Language (BIM) into text in real-time.



🧠 How it Works

Holistic Tracking: Uses MediaPipe to capture 543 landmarks (hands, pose, and face) at 30 FPS.



Normalized Data: Keypoints are anchored to the base of the neck to ensure the model works regardless of where you stand in the frame.



LSTM Model: A Long Short-Term Memory network analyzes the sequence of movements to predict words with high confidence.



🛠️ Quick Start

Install Dependencies:

Open your terminal and run: pip install -r requirements.txt



Launch the Engine:

Run the script using: python live\_translator\_pro.py
